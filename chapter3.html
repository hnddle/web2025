자바스크립트 소개

자바스크립트란?

  • 웹 브라우저에서 동작하는 프로그래밍 언어
  • HTML과 CSS로 만든 정적인 웹페이지를 동적으로 만들어줌
  • 1995년 브렌던 아이크가 개발
  • 처음 이름은 LiveScript → JavaScript로 변경

자바스크립트 특징

  • 인터프리터 언어 : 컴파일 과정 없이 바로 실행
  • 동적 타입 언어 : 변수의 타입을 미리 선언할 필요 없음
  • 객체 기반 언어
  • 함수형 프로그래밍 지원
  • 클라이언트 사이드 스크립트 언어

자바스크립트 사용 방법

1. HTML 내부에 작성

<script>
    alert('안녕하세요!');
</script>

2. 외부 파일로 작성

<script src="script.js"></script>

3. 인라인 방식 (권장하지 않음)

<button onclick="alert('클릭!')">클릭하세요</button>

기본 문법

변수 선언

var name = "홍길동";    // 구버전 방식
let age = 25;          // 블록 스코프 변수
const PI = 3.14;       // 상수

데이터 타입

  • 숫자형 (Number) : 10, 3.14
  • 문자열 (String) : "안녕", '하세요'
  • 불린 (Boolean) : true, false
  • undefined : 값이 할당되지 않은 변수
  • null : 의도적으로 비어있음을 표현
  • 객체 (Object) : {name: "홍길동", age: 25}
  • 배열 (Array) : [1, 2, 3, 4, 5]

출력 방법

console.log("콘솔에 출력");           // 개발자 도구 콘솔
document.write("화면에 출력");         // HTML에 직접 출력
alert("경고창 출력");                  // 경고창

간단한 예제

<script>
    // 변수 선언
    let name = "김철수";
    let age = 20;
    
    // 출력
    console.log("이름: " + name);
    console.log("나이: " + age);
    
    // 조건문
    if (age >= 20) {
        console.log("성인입니다");
    } else {
        console.log("미성년자입니다");
    }
</script>
